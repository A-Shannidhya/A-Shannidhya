## 🌐 Featured Project: [NeptuneBank.online](https://neptunebank.online)

> A fully functional banking system backend with secure user management, microservice architecture, and Kafka integration

### 🔑 Highlights
- ✅ JWT authentication with role-based access control  
- 🔄 Kafka-based event streaming for transactions and logging  
- 🧱 Modular microservices for auth, user, transaction, and OTP  
- 🛠️ Dockerized with CI/CD pipelines via GitHub Actions  
- ☁️ Deployed on AWS EC2 + Nginx under a custom domain  

### 🛠️ Stack Used
Java • Spring Boot • Apache Kafka • MySQL • PostgreSQL • Docker • Gradle • GitHub Actions • RHEL • AWS
